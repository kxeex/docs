## GET /api/veo/status/{task_id}

Check the current status of your video generation task.

<ParamField path="task_id" type="string" required>
  Task UUID returned from the generate endpoint
</ParamField>

<Accordion title="Status Values">
  <ResponseField name="pending" type="string">
    Task is in queue waiting to be processed
  </ResponseField>
  <ResponseField name="processing" type="string">
    Video is currently being generated
  </ResponseField>
  <ResponseField name="completed" type="string">
    Video generation completed successfully
  </ResponseField>
  <ResponseField name="failed" type="string">
    Video generation failed with error
  </ResponseField>
</Accordion>

<CodeGroup>

```bash cURL
curl -X GET https://api.runblob.com/api/veo/status/a1b2c3d4-e5f6-7890 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://api.runblob.com/api/veo/status/a1b2c3d4-e5f6-7890",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
```

</CodeGroup>
